SVG and Icon Resources

A curated collection of SVG and icon libraries, categorized for easy reference.

Popular Icon Libraries

  • Font Awesome - One of the most popular icon libraries, offering thousands of free and pro icons in various styles.
  • Bootstrap Icons - An open-source icon library by Bootstrap's creator, Mark Otto, with SVG-based icons.
  • Ionicons - Clean, MIT-licensed SVG icons, frequently used for web and mobile apps.
  • CSS.gg - An open-source icon library made entirely from CSS, with options for SVG sprites, single icons, and various integrations.

SVG and Icon Search Engines

  • SVGL - Uses a favicon API to retrieve icons from popular sites, ideal for quick access to high-quality favicons.
  • SVG Repo - A search engine for SVG icons, making it easy to find specific icons from multiple categories.
  • Free SVG Icons - A recently launched platform offering free SVG icons.
  • Icons8 - A versatile icon search engine offering icons in multiple styles and formats.
  • The Noun Project - A large library of icons and symbols contributed by designers worldwide.

Framework-Specific and Customizable Icon Libraries

  • Iconify - Offers SVG icons from various collections, organized by site or project for easy integration.
  • Eva Icons - A customizable icon set perfect for web and mobile applications, open-source and easy to use.
  • Feather Icons - A minimalistic, open-source icon library, with a React component option for custom SVG integration.
  • Material Design Icons - A community-enhanced collection with over 5,000 icons for a wide range of uses.
  • Heroicons - Free MIT-licensed icons by Steve Schoger, available in multiple styles for various design needs.
  • Icofont - Offers a CSS-based icon library with easy integration via web fonts.
  • Remixicon - An icon library designed on a 24px grid, offering clean SVG icons for professional use.

SVG Tools and Custom Options

  • Icones - A vast SVG collection that allows users to explore and organize icons based on collections or themes.
  • Flaticon - A large library for free and premium SVG icons, with customizable features for web design projects.
  • CSS.gg 2.0 - Recently updated with even more options, including SVG sprites, Figma and Adobe XD components, and versatile CSS integration.

Animated SVG Libraries

  • Loaf SVG Animations - 200 free animated SVGs available for free with no attribution. Includes a wide range of premium animated icons($99).
